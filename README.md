## Controls

###Debug
* 1: start level 1
* 2: start testing level
* R: respawn
* F9: show/hide colliders
* F10: godmode
* F11: frame cap 60fps

###Gameplay
* A: left
* D: right
* SPACE: jump
* SPACE while on air: double jump
* LEFT-SHIFT: dash with 1s cooldown
* W,A,S,D: godMode movement
